# Amethyst Browser

### About
I started it, because I was dissatisfied with Arc and wanted something that helped my workflow more and was faster. Amethyst is currently pre-release, so there are many planned features missing and bugs may occur. 

Join the Early Access or Waitlist at [Amethyst's Website](https://amethystbrowser.de)

### Key Features
* (customizable) keyboard shortcut focus for quicker navigation
* local search engine (meilisearch) for relevant search suggestions 
* persistent tabs across sessions, including most recent navigation
* very fast compared to e.g. Arc
* hardware security key support (coming soon)
* UI customization (planned)
* Quick access "hotbar" for frequently used sites (planned)
* integrated password manger (planned)
* Ollama integration for summarys and chats (planned)

### Tech-Stack
* Swift
* SwiftUI
* SwiftData
* WebKit
* meilisearch

### Modifications and Redistribution

If you encounter something Amethyst doesn't currently support or you want it done in another way, feel free to make a fork of it. But please make the changes available for others and us to use. 

Amethyst and it's Code are free to use and modify, use for AI training explicitly **excluded**.

If you plan on redistributing Amethyst as a binary, please only do so if you have made relevant changes to it and attribute the core project. Also every redistributed version must be published under the same license as the core amethyst project(AGPLv3, see more in LICENSE file).
